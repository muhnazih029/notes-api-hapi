# Notes API - Backend

A RESTful API for note management built with a clean layered architecture, strict data validation, and interactive documentation.

## 🚀 Built With

- **Framework:** Hapi.js
- **Validation:** Joi
- **Documentation:** Swagger OpenAPI (hapi-swagger)
- **Environment:** Node.js

## 📦 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- [Node.js](https://nodejs.org/) installed on your local machine.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/muhnazih029/notes-api-hapi
```

2. Navigate into the project directory:

```bash
cd /notes-api-hapi
```

3. Install NPM packages:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

## 📖 API Documentation

Once the server is running, the interactive API documentation (Swagger UI) can be accessed through your web browser at:

`http://localhost:5000/docs`
